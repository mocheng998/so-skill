# Appendix: Codes & Reference

## Global Error Codes
| Code | Description | Troubleshooting |
| --- | --- | --- |
| 200 | Success | Success |
| 1600001 | Invalid API key or access token | Get new access token. |
| 1600002 | Access token cannot be empty | Input the correct token. |
| 1600003 | Invalid Refresh token | Use correct Refresh token or get new access token. |
| 1600200 | Too much request | Regulate the rate of your requests. |
| 1600201 | Quota has been used up | Regulate the rate of your requests. |
| 1600300 | Param error | Check your parameters and enter correct value. |
| 1600101 | Interface not found | Check URL. |
| 1602001 | Product not found | Product not found. |
| 1603000 | Order create fail | Order create fail. |
| 1604000 | Balance is insufficient | Top up your CJ balance. |
| 1605000 | Logistic not found | Check shipping method. |
| 1606001 | Webhook setting fail | URL must be HTTPS, POST, 200 OK within 3s. |
