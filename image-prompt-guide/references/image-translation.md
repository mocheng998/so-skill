# 图片文字翻译 (Image Translation)

## 场景说明

将图片中可见的文本全部翻译成目标语言，并直接在图片上将原文替换为译文，生成一张翻译后的新图片。最终效果与原图完全一致，仅文字语言不同。

> **⚠️ 强制约束：本场景必须调用 `image_edit` 工具输出一张翻译后的图片。严禁以文本形式输出翻译结果，严禁跳过工具调用直接回复译文。即使用户没有明确要求"生成图片"，也必须生成图片。**

## 应用方式：直接应用

使用以下 prompt 模板，将 `<target_lang>` 替换为用户指定的目标语言。

## Prompt 模板

```
将提供的图片中可见的文本全部翻译成<target_lang>，并将原文全部替换为译文。最终效果必须与原图完全一致，无法区分。
必须严格遵守
## 字体匹配：
- 完全复制原文的字体、字重、大小、颜色和字距。
- 完美匹配应用于原文字符的任何透视变形、曲率或扭曲效果。
## 背景完整性：
- 保留原始图片背景，不得进行任何更改。
- 不得改变产品或场景的形状、结构或构图。
## 无缝融合：
- 确保新文本与背景自然融合。无可见边缘、切割痕迹或色块差异。
- 模拟逼真的光影效果：新文本必须投射/接收与场景光源一致的阴影。
## 内容准确性：
- 译文必须语法正确且符合上下文语境。
- 请勿修改目标区域以外的任何文本。
- 不得出现乱码、拼写错误或错别字。
```

其中 `<target_lang>` 替换为用户指定的目标语言（如 English、日本語、한국어 等）。

## 工具调用

- 工具：`image_edit`
- task_type：`simple_generation`

> **执行强制规则**：Agent 在匹配到 Image Translation 场景后，必须立即调用 `image_edit` 工具执行图片翻译，禁止以下行为：
> 1. 仅提取图中文字输出翻译文本
> 2. 先输出文本翻译再询问用户是否需要生成图片
> 3. 跳过工具调用直接回复任何形式的译文内容

## 注意事项

- **输出必须且只能是翻译后的图片，严禁输出翻译文本**
- 必须由用户指定目标语言
- 字体、字重、大小、颜色、字距需与原文完全匹配
- 背景不得有任何改变
- 光影效果需与场景一致
- 译文需语法正确且符合上下文语境
- 不得修改目标区域以外的文本
