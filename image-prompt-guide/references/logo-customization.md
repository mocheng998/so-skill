# Logo 定制 (Logo Customization)

## 场景说明

将用户提供的 Logo 图片合成到商品图上，使用指定的工艺方式，智能放置在符合工艺特性和商品品类惯例的合适位置。

## 应用方式：拼接

需要选择定制工艺（craft_prompt），然后与固定合成 prompt 拼接。

## Prompt 模板

### 第一步：选择定制工艺（craft_prompt）

必须从以下列表中选择一项：

- 激光雕刻
- 丝网印刷
- 热转印
- 激光打印
- 贴纸 / 转印贴
- 烫金 / 烫印
- UV 打印
- 数码印刷
- 刺绣 / 绣花
- 3D 打印
- 压纹 / 凸印 / 凹印
- 喷墨打印
- 常规/彩色印刷
- 单色 / 双色印刷

### 第二步：拼接完整 prompt

```
图2中的整个Logo 合成到图1主体上，应用"{craft_prompt}"，并智能放置在符合该工艺特性和商品品类惯例的合适位置。仅保留 Logo 本身，去除logo周围的背景或边缘，实现干净透明的融合。
```

其中 `{craft_prompt}` 替换为上方选择的工艺名称。

### 示例

若选择"烫金 / 烫印"，完整 prompt 为：

```
图2中的整个Logo 合成到图1主体上，应用"烫金 / 烫印"，并智能放置在符合该工艺特性和商品品类惯例的合适位置。仅保留 Logo 本身，去除logo周围的背景或边缘，实现干净透明的融合。
```

## 工具调用

- 工具：`image_edit`
- task_type：`simple_generation`
- 输入：需要两张图片——图1（商品图）和图2（Logo 图）

## 注意事项

- 工艺选择需由用户指定或 Agent 根据商品品类智能推荐
- Logo 周围的背景或边缘需去除，实现干净融合
- 放置位置需符合所选工艺特性和商品品类惯例
