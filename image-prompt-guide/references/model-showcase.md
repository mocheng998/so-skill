# 模特图 (Model Showcase)

## 场景说明

基于商品图片生成高质量电商模特展示图，让模特自然展示商品的穿戴或使用效果。根据用户输入情况分三种变体处理。

## 应用方式：直接应用

根据以下优先级判断使用哪个变体：

1. **变体 A**：用户上传了模特参考图 → 直接使用用户上传的模特图
2. **变体 B**：用户未上传模特图，但指定了模特类型（如"亚洲女性模特"） → 将 `<model_type>` 替换为用户描述
3. **变体 C**：用户既未上传模特图，也未指定模特类型 → 使用固定文案"根据商品图添加适配商品场景的模特图"

## Prompt 模板

### 变体 A：有模特参考图

用户同时提供了商品图和模特图：

```
请基于上传的商品图片和模特图片生成一张高质量电商模特展示图：

## 生成要求：
- 生成的模特展示图图片的背景必须和商品图的背景完全一致。
- 商品必须与商品图完全一致，不改变颜色、结构、比例、纹理或细节。
- 模特需自然展示商品，使商品成为画面主体，展示商品的穿戴或使用效果。
- 生成的模特展示图图片保留模特的头部。
- 商品主体和模特的比例要协调，不要出现认知冲突。

## 根据商品类型合理安排展示方式，例如：
- 服装/鞋靴/运动户外鞋服及装备类：模特自然穿着展示整体版型和细节。
- 配饰/珠宝首饰/箱包类：模特佩戴展示，突出商品细节。
- 家居园艺/家具/家用电器/医疗器械/礼品工艺品/运动娱乐/办公文教用品/包装印刷/母婴&玩具/工业机械/消费电子/美妆/宠物用品/个人护理家庭清洁类：模特在真实生活场景中自然使用或互动展示商品。
```

## 工具调用

- 工具：`image_edit`
- task_type：`simple_generation`

## 注意事项

- 变体选择优先级：A（有模特图）> B（有模特类型描述）> C（均无）
- 商品的颜色、结构、比例、纹理和细节不得改变
- 背景必须与商品图背景保持一致
- 必须保留模特头部
- 根据商品品类自动匹配展示方式（穿着、佩戴、使用场景等）
