# SKU 换色 (SKU Color Change)

## 场景说明

对商品进行颜色替换，严格保持原有款式、轮廓、纹理结构及光照效果不变，实现自然逼真的换色效果。

## 应用方式：拼接

在用户或 Agent 根据需求生成的换色描述 prompt **后面**拼接以下固定文案。

## Prompt 模板

### 用户/Agent 生成部分

由用户描述期望的目标颜色，例如："Change the product color to navy blue" 或 "将商品颜色改为酒红色"

### 固定拼接文案

```
严格保持其原有的款式、轮廓、纹理结构及光照效果不变。颜色替换需自然逼真，避免出现不协调的色偏或光影失真，同时完整保留图像中的所有非主体细节（如背景、配饰、材质反光等），确保整体视觉效果真实且一致。
```

### 完整 prompt 结构

```
{用户/Agent 的换色描述 prompt} + 严格保持其原有的款式、轮廓、纹理结构及光照效果不变。颜色替换需自然逼真，避免出现不协调的色偏或光影失真，同时完整保留图像中的所有非主体细节（如背景、配饰、材质反光等），确保整体视觉效果真实且一致。
```

## 工具调用

- 工具：`image_edit`
- task_type：`simple_generation` 或 `complex_generation`（根据换色复杂度判断）

## 注意事项

- 款式、轮廓、纹理结构必须严格保持不变
- 光照效果需与原图一致，避免色偏或光影失真
- 背景、配饰、材质反光等非主体细节必须完整保留
