# Module 1: Brand Understanding - Execution Guide

## Execution Steps

### Step 1: Research Brand Information

Use `info_search` to gather:
- Official website content
- Company description
- Social media positioning
- Recent news/press releases

### Step 2: Extract Context

Identify from research + user query:
- **Industry:** Tech, Fashion, F&B, etc.
- **Target Market:** B2B/B2C, Premium/Mass
- **User Persona:** Who are the customers?
- **Core Value:** What problem does the brand solve?
- **Personality:** Professional, playful, innovative?

### Step 3: Analyze Existing Logo (if provided)

Use `image_analysis` to parse:
- Visual elements and symbolism
- Current color palette
- Typography style

### Step 4: Synthesize Insights

Combine findings into actionable design direction.

---

## Output Format

```
**Brand Insight:**
[Brand] targets [audience] in [industry]. Core value: [proposition]. 
Aims to convey [message] through [usage scenarios].

**Existing Visual Language** (if applicable):
Current logo uses [elements] symbolizing [meaning], with [colors] reflecting [traits].

**Recommended Design Direction:**
Logo should embody [principles] to align with [values] and resonate with [audience].
```

---

## Example Output

```
**Brand Insight:**
NexCore targets enterprise developers in B2B tech. Core value: reliable AI infrastructure.
Aims to convey innovation and stability.

**Recommended Design Direction:**
Balance futuristic aesthetics with geometric stability. Use clean lines and 
modern palette (tech blues/grays) to appeal to technical decision-makers.
```
