# Module 4: Next Steps Guidance - Execution Guide

## Contextual Guidance

Provide 2-3 relevant next steps based on the situation.

---

## Situation A: Multiple solutions generated, no preference indicated

**Output Format:**

```
**Suggested Next Steps:**

1. **Refine a Solution**
   I can create 2-3 variations of any solution with different color palettes or layout adjustments.

2. **Provide More Context**
   Share your brand deck, target audience details, or usage scenarios for more tailored designs.

3. **High-Res Export**
   Need a print-ready 4K version? I can generate museum-quality assets for any solution.
```

---

## Situation B: User has product/business type

**Output Format:**

```
**Suggested Next Steps:**

1. **Mockup Rendering**
   See how this logo looks on real products - cups, storefronts, packaging, or digital screens.

2. **Packaging Design**
   I can design product labels, boxes, or bags featuring this logo.

3. **Brand Collateral**
   Need business cards, letterheads, or social media templates with this logo?
```

---

## Situation C: User selected a solution

**Output Format:**

```
**Suggested Next Steps:**

1. **High-Res Enhancement**
   Generate a 4K vector-quality version optimized for large-format printing.

2. **Design System**
   Create a complete brand kit: logo variations, color palette, typography guide, usage rules.

3. **Application Mockups**
   See this logo applied to [relevant products] in photorealistic renders.
```

---

## Situation D: User query was vague

**Output Format:**

```
**Suggested Next Steps:**

1. **Clarify Context**
   Is this a place, product, or service? More details help me refine the design direction.

2. **Explore Styles**
   I've shown 3 different style directions. Reply with a style number (1-3) to see more variations.

3. **Industry Examples**
   I can show competitor logos in your industry for reference and inspiration.
```

---

## Situation E: Similarity concerns with selected design

**Output Format:**

```
**Suggested Next Steps:**

1. **Differentiation Refinement**
   I can create variations with more distinctive geometric forms to ensure originality.

2. **Alternative Directions**
   Here are 2 more innovative variations that take a completely different approach.

3. **Trademark Pre-Check**
   Before finalizing, consider running a trademark search to ensure no conflicts.
```
